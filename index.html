Page Replacement Algorithms

FIFO

First-In-First-Out replaces the oldest page in memory, regardless of how often it's been used.

  • Simple implementation with a queue
  • Low overhead
  • May replace frequently used pages
Learn more

LRU

Least Recently Used replaces the page that hasn't been accessed for the longest time.

  • Approximates optimal algorithm
  • Better performance than FIFO
  • Higher implementation overhead
Learn more

ESC

Enhanced Second Chance combines FIFO with reference and modify bits for better decisions.

  • Considers page usage patterns
  • Better than pure FIFO
  • Moderate implementation complexity
Learn more
